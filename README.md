# Jarvis AI Assistant 🤖

Jarvis is a Python-based voice assistant powered by OpenAI's GPT model. It can respond to voice commands, perform basic tasks, and generate AI-based responses using natural language processing.

---

## 🚀 Features

- 🎤 Voice recognition (speech to text)
- 💬 GPT-powered chat responses
- 📂 File and web integration
- 🧠 Remembers conversation context
- 🔐 Secure environment setup (API keys via `.env`)
- 🎧 Text-to-speech replies (Jarvis talks back)

---

JarvisAI/

env
